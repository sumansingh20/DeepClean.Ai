# Frontend

## Environment

- Node.js 18+
- npm or yarn or pnpm

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:3000`

## Building

```bash
npm run build
npm start
```

## Linting & Formatting

```bash
npm run lint
npm run format
```

## Testing

```bash
npm test
npm run test:watch
npm run test:coverage
```

## Type Checking

```bash
npm run type-check
```
